### 算法题复习
1. 反转链表 
```python  
# 递归方法    

def reverse(head):
    if not head:
        return   
    last = reverse(head.next) 
    head.next.next = head 
    head.next = None 
    return last 

# 迭代方法 
# 反转链表两个指针不够, 需要三个指针, pre, cur, nxt  
pre = None 
cur = head.next 
while cur:  
    nxt = cur.next # 暂存原next, 否则会丢失  
    cur.next = pre # 反转 
    pre = cur # 新头变旧头 
    cur = nxt # 暂存的旧头 变 新头 
```
