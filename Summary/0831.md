1. 二叉树的前序遍历
```python  
# 递归方法    
if not root: return [] 
res = [] 

def dfs(root):
    # base case
    if not root: 
        return [] 
    res.append(root.val) 
    dfs(root.left) 
    dfs(root.right)  

dfs(root) 
return res

# 需要用到栈  

```
