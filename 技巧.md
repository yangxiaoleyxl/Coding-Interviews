## 技巧题 （From Hot 100）  
### Hot 100 中收录的一些经典技巧题 
该部分涉及到数学技巧，包含的内容有：
- 位运算
- 摩尔投票法

- [ Single Number ]( https://leetcode.cn/problems/single-number/description/ )  
```python 
class Solution:
    def singleNumber(self, nums: List[int]) -> int:  
        # 使用异或运算, 异或的性质：任何数和自己做异或为 0 
        # 执行一次全员的异或即可, 位运算技巧
        singleNum = 0
        for num in nums:
            singleNum ^= num 
        return singleNum 
```
